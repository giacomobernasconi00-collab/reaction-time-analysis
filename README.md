# reaction-time-analysis
Analysis of reaction times under stress vs control conditions
#  Reaction Time Analysis

This project explores differences in reaction times between a control group and a stress condition.

##  Methods
- Simulated dataset (N = 200)
- Two groups: control vs stress
- Statistical test: independent t-test

##  Results
The stress group showed slower reaction times compared to the control group.

##  Tools
- Python
- Pandas
- Seaborn
- SciPy
